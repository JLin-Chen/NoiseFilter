# NoiseFilter  
本篇文章将会涵盖下述几个方面：  
1. 讲解3种基本的噪声，包括高斯噪声、均匀分布噪声和椒盐噪声，的形成原因以及它们的概率密度函数。  
2. 掌握使用matlab模拟生成噪声的操作，探明不同参数的噪声如何干扰图像。  
3. 对10种典型的滤波器，包括算数均值滤波、几何均值滤波、谐波均值滤波、逆谐波均值滤波、中值滤波器、最大值滤波器、最小值滤波器、
中点滤波器、修正后的alpha均值滤波器、自适应中值滤波器，滤除噪声的能力进行探究和比较。
